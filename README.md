## ✨ Features

- **Neo-Brutalist** design — hard borders, bold typography, no gradients, raw honesty
- **GSAP animations** — hero name reveal, scroll-triggered reveals, staggered card entrances
- **Text scramble effect** — hover the hero name for a glitch scramble
- **Custom cursor** — shape-shifting triangle/diamond/circle cursor
- **Animated skill bars** — scroll-triggered progress fills
- **Dual marquee tickers** — scrolling tech stack and motto
- **Counter animations** — live-counting stats on hero load
- **Scroll progress bar** — red progress indicator at top
- **Keyboard navigation** — press 1–5 to jump between sections
- **Loading screen** — typewriter loader with progress bar
- **Fully responsive** — mobile hamburger menu, fluid typography
- **Contact form** — with animated submit feedback
- **Noise texture overlay** — subtle grain for depth
- **MIT licensed** — use it, fork it, build on it

## 🚀 Local Development

```bash
# Clone
git clone https://github.com/shaikhshahnawaz13/portfolio.git
cd portfolio

# Install dev tools (optional)
npm install

# Start dev server
npm run dev
# → Opens at http://localhost:3000 with live reload
```

Or just open `index.html` directly in a browser — zero build step required.
